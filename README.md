# NodeJS API Project

## Description
This is a simple Node.js API created for DevOps assignment.

## Features
- GET / → Welcome message
- GET /foods → List of foods
- POST /order → Create order

## Technologies Used
- Node.js
- Express.js

## How to Run
1. Install dependencies:
   npm install

2. Start server:
   node index.js

3. Open browser:
   http://localhost:3000
